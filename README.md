# Class Repository

This is the class repository for the Java Full Stack Upskilling course running June 1, 2026, to July 3, 2026.

You will not be able to access this repository from you work computers. However, all the slides and labs will also be distributed to you directly through your training coordinator. 

You can also access the materials from your personal devices.

This repository will also have a number of additional resources for further study and any materials that were added as a result of the classroom work.

### Availability

The repository will be available for you to download or clone for 30 days after the end of the class. The repository will be archived on August 4, 2026.

### Lab Repository

There is a copy of the Labs folder in a separate Labs repository located [here](https://github.com/ExgnosisClasses/2612-JavaSpring-Labs).

This will allow you to clone the labs into your provided lab virtual machine, or to access the labs in a browser in your VM, without having to clone all the rest of the content into your VM.

### Change Log

There is a separate ChangeLoge file located in the top level of this repository where all changes to the repository will be recorded.
