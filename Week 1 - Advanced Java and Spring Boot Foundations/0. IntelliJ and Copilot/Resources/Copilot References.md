# GitHub Copilot: References and Further Reading

**Java Full Stack Development**

---

## Official Documentation

The resources below are drawn from the official GitHub documentation site and the JetBrains Marketplace. They cover installation, core usage, and the chat and prompt engineering features introduced in this course.

---

### GitHub Copilot Docs Home

The main documentation hub for all Copilot features across supported editors, plans, and configurations. The best starting point for any question not answered in course materials.

[docs.github.com/en/copilot](https://docs.github.com/en/copilot)

---

### What is GitHub Copilot?

An overview of how Copilot works, what plans are available, and what capabilities are included at each tier. Useful background reading before or alongside the Copilot module.

[docs.github.com/en/copilot/about-github-copilot/what-is-github-copilot](https://docs.github.com/en/copilot/about-github-copilot/what-is-github-copilot)

---

### Copilot Quickstart

A guided introduction to getting Copilot working in your environment. Covers account setup, plugin installation, and your first inline suggestion. Recommended reading for students who want to get oriented quickly before the lab sessions.

[docs.github.com/en/copilot/quickstart](https://docs.github.com/en/copilot/quickstart)

### Installing GitHub Copilot in IntelliJ IDEA

Step-by-step instructions for installing the GitHub Copilot plugin from the JetBrains Marketplace, authenticating with your GitHub account, and verifying the integration is working inside IntelliJ IDEA.

[docs.github.com/en/copilot/managing-copilot/configure-personal-settings/installing-the-github-copilot-extension-in-your-environment](https://docs.github.com/en/copilot/managing-copilot/configure-personal-settings/installing-the-github-copilot-extension-in-your-environment)

The official plugin page, including the current version, changelog, user reviews, and installation instructions. Check here for compatibility information if you run into issues after an IntelliJ IDEA update.

### Using GitHub Copilot in Your IDE

Covers inline code completions in detail: how suggestions are triggered, how to accept or dismiss them, how to cycle through alternatives, and the keyboard shortcuts covered in the course module.

[docs.github.com/en/copilot/using-github-copilot/using-github-copilot-in-your-environment](https://docs.github.com/en/copilot/using-github-copilot/using-github-copilot-in-your-environment)

### Copilot Chat

Explains the conversational chat panel, including how to use slash commands such as `/explain`, `/fix`, `/tests`, and `/doc`. Also covers how Copilot Chat uses the currently open file and selected code as context for its responses.

[docs.github.com/en/copilot/github-copilot-chat/using-github-copilot-chat-in-your-ide](https://docs.github.com/en/copilot/github-copilot-chat/using-github-copilot-chat-in-your-ide)

### Prompt Engineering for GitHub Copilot

Guidance on writing comments and structuring code to produce more accurate and useful suggestions from Copilot. Understanding this material helps students move beyond passive acceptance of completions toward actively directing Copilot toward the output they want.

[docs.github.com/en/copilot/using-github-copilot/prompt-engineering-for-github-copilot](https://docs.github.com/en/copilot/using-github-copilot/prompt-engineering-for-github-copilot)

## Additional Readings and References for Deeper Study

The resources below go beyond the course material and are recommended for students who want to build a deeper understanding of how Copilot works, how to use it more effectively, and where it is heading.

---

### GitHub Blog: Copilot

The official GitHub blog posts related to Copilot, covering new feature announcements, case studies, and guidance from the engineering team. A useful resource for staying current as the tool evolves rapidly.

[github.blog/tag/github-copilot](https://github.blog/tag/github-copilot/)

---

### Best Practices for Using GitHub Copilot

Official guidance on getting the most out of Copilot across different development tasks. Includes specific advice on Java workflows, writing effective prompts, and validating generated code before committing it.

[docs.github.com/en/copilot/using-github-copilot/best-practices-for-using-github-copilot](https://docs.github.com/en/copilot/using-github-copilot/best-practices-for-using-github-copilot)

### GitHub Copilot for Java Developers on Microsoft Learn

A practical learning path walking through Copilot-assisted development with Java and Spring Boot. Covers code generation, test generation, and refactoring scenarios that map directly to lab work in this course.

[learn.microsoft.com/en-us/training/paths/copilot](https://learn.microsoft.com/en-us/training/paths/copilot/)

---

### Copilot App Modernization for IntelliJ IDEA *(GitHub Changelog, October 2025)*

Introduces the app modernization capabilities added to the IntelliJ IDEA Copilot plugin in late 2025, including automated code transformations, dependency patching, CVE remediation, and containerization. Relevant for students working on upgrading or migrating existing Java codebases.

[github.blog/changelog/2025-10-28-github-copilot-app-modernization-now-available-in-intellij-idea](https://github.blog/changelog/2025-10-28-github-copilot-app-modernization-now-available-in-intellij-idea/)

---

### The Illustrated Transformer by Jay Alammar *(Background Reading)*

A conceptual, visual explanation of how transformer-based large language models work. Reading this helps demystify what Copilot is doing under the hood: how it interprets context, why it sometimes produces plausible-looking but incorrect suggestions, and what its fundamental limitations are. No prior machine learning background is required.

[jalammar.github.io/illustrated-transformer](https://jalammar.github.io/illustrated-transformer/)

---

> **Note for Students:** Begin with the Quickstart and the installation guide to get Copilot working in IntelliJ IDEA. Then read the prompt engineering documentation before the lab sessions, as knowing how to direct Copilot intentionally is more valuable than accepting its first suggestion by default. The additional readings are best revisited after you have spent time using the tool and have formed your own questions about how it works.
