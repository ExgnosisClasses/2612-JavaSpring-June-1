# IntelliJ IDEA: References and Further Reading

**Java Full Stack Development**

---

## Official Documentation

The resources below are drawn from the JetBrains documentation site and cover the features and workflows used throughout this course. Each link points directly to the relevant section of the official docs.

---

### IntelliJ IDEA Docs Home

The main documentation hub covering all IDE features, settings, and supported languages and frameworks. Start here if you are unsure where to find something.

[jetbrains.com/help/idea](https://www.jetbrains.com/help/idea/)

---

### Getting Started Guide

Covers installation, initial project setup, and the core concepts you need to become productive in the IDE quickly. Recommended reading before or alongside the first week of the course.

[jetbrains.com/help/idea/getting-started.html](https://www.jetbrains.com/help/idea/getting-started.html)

---

### Keyboard Shortcuts Reference

Full shortcut cheat sheets for Windows/Linux and macOS. Familiarity with these shortcuts is one of the fastest ways to improve day-to-day productivity in the IDE.

[jetbrains.com/help/idea/reference-keymap-win-default.html](https://www.jetbrains.com/help/idea/reference-keymap-win-default.html)

---

### Debugging in IntelliJ IDEA

Covers breakpoints, stepping through code, watches, the evaluate expression tool, and the debug tool window. Closely aligned with the debugging module in this course.

[jetbrains.com/help/idea/debugging-code.html](https://www.jetbrains.com/help/idea/debugging-code.html)

---

### Running Tests

Explains JUnit and TestNG integration, how to run individual tests or full test suites, and how to view test coverage reports from within the IDE.

[jetbrains.com/help/idea/testing.html](https://www.jetbrains.com/help/idea/testing.html)

---

### Maven Projects

Covers importing Maven projects, working with the pom.xml file, and enabling auto-import so the IDE stays in sync with dependency changes without manual intervention.

[jetbrains.com/help/idea/maven-support.html](https://www.jetbrains.com/help/idea/maven-support.html)

---

### Code Inspections and Refactoring

Describes the built-in static analysis tools, how to act on quick-fix suggestions, and refactoring operations such as rename, extract method, and inline variable.

[jetbrains.com/help/idea/code-inspection.html](https://www.jetbrains.com/help/idea/code-inspection.html)

---

### File Encoding Settings

Explains how to configure UTF-8 across source files, resource files, and configuration files. Particularly important on Windows machines where the system default may differ from what Java and Spring Boot expect.

[jetbrains.com/help/idea/configuring-file-encoding.html](https://www.jetbrains.com/help/idea/configuring-file-encoding.html)

---

### Spring Boot in IntelliJ IDEA

Covers creating Spring Boot projects using the built-in wizard, running and debugging Spring Boot applications, and working with the Spring tool window.

[jetbrains.com/help/idea/spring-boot.html](https://www.jetbrains.com/help/idea/spring-boot.html)

---

### What's New in IntelliJ IDEA (2026.1)

The latest release overview, including AI agent integrations, Spring runtime insight, quota-free next-edit suggestions, and Java and Kotlin improvements.

[jetbrains.com/idea/whatsnew](https://www.jetbrains.com/idea/whatsnew/)

## Additional Readings and References for Deeper Study

The resources below go beyond the course material and are recommended for students who want to build a deeper or more rounded understanding of IntelliJ IDEA.

---

### JetBrains IDE Tips and Tricks Blog

The official JetBrains blog for IntelliJ IDEA publishes ongoing articles, video walkthroughs, and deep dives into productivity features, lesser-known shortcuts, and new capabilities with each release. A useful resource to follow throughout the course and beyond.

[blog.jetbrains.com/idea](https://blog.jetbrains.com/idea/)

---

### IntelliJ IDEA on YouTube

The official JetBrains YouTube presence includes tutorial videos on topics such as debugging, refactoring, Spring Boot, testing, and Maven integration. Video format can be helpful for seeing IDE workflows in motion rather than reading about them.

[youtube.com/@intellijidea](https://www.youtube.com/@intellijidea)

---

### IntelliJ IDEA Keymaps (Printable PDF)

Downloadable and printable shortcut reference cards for both Windows/Linux and macOS. Useful to keep nearby during the early weeks of the course while shortcut muscle memory is still forming.

[jetbrains.com/idea/resources](https://www.jetbrains.com/idea/resources/)

---

### "IntelliJ IDEA Essentials" by Jaroslaw Krochmalski *(Book)*

A focused book on productive use of the IDE. Covers the IntelliJ project model, version control integration, build tool configuration, and advanced navigation and editing features. A good reference for students who prefer reading comprehensive coverage in one place.

---

### JetBrains Academy: IntelliJ IDEA Track

Interactive, hands-on exercises hosted on Hyperskill that teach the IDE progressively from the ground up. Useful for students who want structured practice outside of class labs.

[hyperskill.org](https://hyperskill.org/)

---

### "Effective Java, Third Edition" by Joshua Bloch *(Book)*

While not an IntelliJ-specific resource, this book is the definitive reference for writing clean, efficient, and idiomatic Java. Reading it alongside the course helps students understand the reasoning behind many of the code quality suggestions IntelliJ surfaces through its inspections and quick-fixes.

---

> **Note for Students:** The official documentation and the JetBrains blog are the most immediately practical resources. Bookmark the docs home and the keyboard shortcut reference on day one. Return to the deeper reading list as you progress through the course and identify specific areas where you want to go further.
